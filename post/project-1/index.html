Sentiment Analysis of Amazon Reviews - MLOps | NLP

April 25, 2024

Project Overview

This project focuses on building a scalable, end-to-end machine learning pipeline for sentiment analysis of Amazon product reviews using Natural Language Processing (NLP) techniques, adhering to Machine Learning Operations (MLOps) principles. The goal is to automatically analyze customer feedback, extract actionable insights, enabling businesses to enhance product development and improve customer service.

Leveraging advanced NLP techniques such as tokenization, lemmatization, sentiment scoring, and thematic modeling, we built a robust and scalable system. We utilized technologies like TF-IDF, word embeddings, LSTM, and BERT models for feature extraction and sentiment classification, ensuring high accuracy and scalability.

Key Achievements

  • Built an end-to-end MLOps pipeline incorporating NLP techniques to process Amazon reviews and extract meaningful insights.
  • Integrated Sentiment Analysis to provide more granular insights into customer satisfaction regarding specific product features.
  • Deployed the entire solution on Google Cloud Platform (GCP) using Vertex AI and Docker to ensure scalability and seamless integration with real-time review data.

Technical Skills Used

  • Programming Languages: Python
  • Libraries: BeautifulSoup, NLTK, Scikit-learn, SpaCy, TensorFlow
  • Tools: Apache Airflow, Data Version Control (DVC), Docker, Git, Jupyter Notebook, VS Code
  • Cloud Services: Google Cloud Storage, Vertex AI

Methodology

  1. Data Collection:

    • Scraped Amazon product reviews using BeautifulSoup to collect diverse feedback data.

  2. Data Preprocessing:

    • Applied tokenization, stop word removal, and lemmatization to clean the text data.

  3. Model Building:

    • Trained two models: LSTM and BERT for sentiment classification.
    • Used TF-IDF and word embeddings for feature extraction to improve model accuracy.

  4. Pipeline Orchestration:

    • Utilized Apache Airflow for orchestrating the machine learning workflow, automating data preprocessing, model training, deployment, monitoring, and retraining.
    • Managed version control of models and data using DVC.

  5. Evaluation:

    • Achieved 85% accuracy on the test set for sentiment classification, ensuring reliable performance in identifying customer sentiment.
    • Measured model performance using ROC-AUC and Precision-Recall metrics.

Results

  • High Accuracy: The sentiment analysis model achieved an 85% accuracy rate, providing reliable insights into customer satisfaction trends.
  • Business Impact: By analyzing product-specific sentiment through aspect-based analysis, businesses can target specific areas for improvement and boost customer satisfaction.
  • Scalable Pipeline: The MLOps framework ensured scalability and ease of integration with cloud platforms for continuous feedback processing.

Future Improvements

  • Real-time Analysis: Further optimization of the pipeline to process real-time review data.
  • Model Enhancements: Fine-tuning the BERT model to improve accuracy for niche product categories.
