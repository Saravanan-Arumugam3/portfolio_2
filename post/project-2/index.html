Impact of Covid-19 on New York City's 3-1-1 Service - Data Visualization | Tableau

December 20, 2022

Project Overview

This project focused on visualizing the impact of the COVID-19 pandemic on New York City’s 3-1-1 municipal service requests using Tableau. By analyzing over 31 million service requests, we developed an interactive dashboard to uncover patterns in complaint types and response times, particularly during the pandemic. The objective was to provide actionable insights for city planners and public officials, aiding in municipal service management and helping improve the efficiency of city agencies in future crises.

Our solution provided an intuitive interface for understanding how the pandemic reshaped the nature and frequency of 3-1-1 requests across the five boroughs, shedding light on the adaptability and responsiveness of the city’s infrastructure.

Key Achievements

  • Analyzed 31 million service requests: Built an efficient data pipeline using Google BigQuery and Tableau Prep to clean, filter, and analyze a large dataset.
  • Interactive Dashboard: Designed a user-friendly dashboard to visualize key complaint patterns and response times across NYC boroughs before and during COVID-19.
  • Insightful Visualizations: Delivered data-driven insights that revealed the shift in service requests during the pandemic, showcasing areas where city services were resilient and areas needing improvement.

Technical Skills Used

  • Programming Languages: Python (for data preprocessing)
  • Libraries: Pandas
  • Tools: Tableau, Tableau Prep, Google BigQuery
  • Cloud Services: Google BigQuery
  • Data Skills: Data Cleaning, Data Visualization, Exploratory Data Analysis (EDA)

Methodology

  1. Data Collection:

    • Pulled NYC 3-1-1 service request data from NYC Open Data’s Socrata API, covering service requests spanning multiple years.

  2. Data Preprocessing:

    • Used Google BigQuery to filter and reduce the dataset from over 31 million records to a more manageable 12 million rows and 25 columns, focusing on key variables.
    • Cleaned the dataset in Tableau Prep by handling missing values and irrelevant columns, ensuring consistency and accuracy. Then grouped the complaint types and created a data pipeline using Tableau Perp’s data flow.

  3. Exploratory Data Analysis (EDA):

    • Conducted a thorough analysis of complaint distribution across NYC boroughs using Tableau.
    • Identified common complaint types and trends, including the shift in service request types during the pandemic and changes in agency response times.

  4. Dashboard Development:

    • Designed two primary dashboards to visualize:
      1. Overall Distribution & Response Times: Showcased trends in complaint types and response efficiency before and during the pandemic.
      2. COVID-19 Impact on Service Requests: Highlighted shifts in service request volumes and identified the pandemic’s influence on city agency performance.

Results

The project focused on visualizing data rather than predictive modeling. Our analysis provided two comprehensive dashboards that offered valuable insights:

  1. Complaint Distribution & Response Times: Illustrated trends in complaint types and response times, helping city planners understand the strain COVID-19 placed on municipal services.
  2. Pandemic Impact Analysis: Revealed how the pandemic affected service request volumes and the adaptability of city agencies, offering a detailed view of system efficiency under crisis conditions.

Key Insights

  • Significant shifts in complaint types were observed during the pandemic, including a rise in noise complaints and public health-related issues.
  • The response times for specific complaint types varied, indicating areas where city services adapted well and areas where there was room for improvement.

Future Scope

  • Predictive Modeling: Implement machine learning models to predict future complaint volumes and trends based on historical data.
  • Real-time Updates: Integrate real-time data to make the dashboard dynamic and provide up-to-date insights on service requests.
  • Enhanced Analysis: Apply advanced machine learning techniques to cluster complaints and identify deeper trends in service efficiency.
